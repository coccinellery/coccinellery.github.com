This is a gallery of semantic patches for use with Coccinelle. They are extracted automatically from a collection of semantic patches that have been used to create patches for the Linux Kernel and other software. The descriptions are derive from the commit messages contained in those patches. There is no guarantee that the descriptions are correct or understandable, and no guarantee that the semantic patches are correct, complete, or work with the latest version of Coccinelle. Comments and corrections may be sent to the Coccinelle mailing list: cocci@systeme.lip6.fr, list info page. To propose new semantic patches to add to the gallery, please make a pull request via Github to Coccinellery repository.
